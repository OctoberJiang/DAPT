# Report

The final output of the DAPT should be a structured report, including the description of the vulnerabilities, complete attack chain, severity, etc.

The user should be able to specify the file format of the report and the path to store it. 

## Sub-Plans

> Naming Criteria: `7_X_<plan-name>`

- `7_1_structured-report-generation`: completed report assembly from repo-local campaign artifacts, machine-readable and human-readable rendering, and output-path control.
