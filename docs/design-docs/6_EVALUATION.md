# Evaluation

The evaluation is split into two parts:

- CTF Mode
- Real-world Mode

For CTF Mode, XBOW-benchmarks are used to evaluate the DAPT. You should add scripts to run the them, providing the user the avability to choose the target benchmark number(one; all; an array of numbers) and the benchmark directory path(default to the benchmarks in the proj).

## Sub-Plans
> Naming Criteria: `6_X_<plan-name>`

- `6_1_xbow-ctf-evaluation-runner`: completed repo-local XBOW benchmark discovery, selection, lifecycle orchestration, and evaluation artifact generation.
